## Objective
Analyze clinical and demographic factors associated with heart disease using exploratory data analysis.

## Dataset
- Source: Kaggle (Heart Disease Dataset)
- Size: 70,000+ records
- Target variable: heart_stroke

## Key Questions
- How does stroke prevalence vary by age and sex?
- Which clinical metrics show the strongest separation between cases and controls?

## Key Insights
- Blood pressure metrics show the strongest association with stroke
- Glucose levels are consistently higher among stroke cases
- Cholesterol shows moderate separation
